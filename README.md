# ItchyVersioning

### A simple, easy-to-use versioning system for Scratch projects

ItchyVersioning is a simple program providing the ability to commit and monitor a Scratch project similarly to Git. 

### Features:

- Commits with messages, authors, and times
- Releases

### Roadmap:

In the future, ItchyVersioning will likely support:

- Rollbacks
- Permissions
- Custom release messages
- Merging, diffing, and patching
- And more
